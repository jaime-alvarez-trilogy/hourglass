Open this page on your iPhone to install.
Open the downloaded file and iOS will ask you to open it with Scriptable. Once imported, run the script to set up your account.

Setup (2 minutes)

1
Tap Download WorkSmart above — then open the downloaded file and choose Open in Scriptable
2
Run WorkSmart in Scriptable — the setup wizard walks you through login and team detection
3
Long-press your Home Screen → tap + → search for Scriptable → add a widget in the size you prefer
4
Long-press the widget → Edit Widget → set Script to WorkSmart and When Interacting to Open App
