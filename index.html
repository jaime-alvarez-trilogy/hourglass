Open this page on your iPhone to install.
Open the downloaded file and iOS will ask you to open it with Scriptable. Once imported, run the script to set up your account.

Setup (2 minutes)

1
Tap Download HourGlass above — then open the downloaded file and choose Open in Scriptable
2
Run HourGlass in Scriptable — the setup wizard walks you through login and team detection
3
Long-press your Home Screen → tap + → search for Scriptable → add a widget in the size you prefer
4
Long-press the widget → Edit Widget → set Script to HourGlass and When Interacting to Open App

Siri Voice Commands

After setup, ask Siri about your Crossover hours, earnings, and pending approvals. Add shortcuts below — one tap each.

"Hey Siri, Crossover summary"
Hours, earnings, deadline, and pending approval count
"Hey Siri, check my hours"
Weekly total, today's hours, remaining to goal
"Hey Siri, Crossover earnings"
Weekly earnings and hourly rate
"Hey Siri, Crossover AI status"
AI usage percentage and BrainLift hours vs targets
"Hey Siri, check approvals" Managers
Pending manual time and overtime requests
"Hey Siri, approve timecards" Managers
Approve all pending requests by voice
